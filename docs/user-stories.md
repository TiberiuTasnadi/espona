# User Stories

## Épica: Gestión de parcelas

* Como agricultor quiero crear una parcela para poder organizar mi explotación.
* Como responsable quiero ver todas las parcelas de una explotación para tener visión global.
* Como usuario quiero consultar los detalles de una parcela para entender su estado.

---

## Épica: Gestión de cultivos

* Como agricultor quiero registrar qué cultivo hay en cada parcela para llevar control.
* Como responsable quiero ver el histórico de cultivos para analizar productividad.
* Como sistema quiero evitar que una parcela tenga más de un cultivo activo.

---

## Épica: Gestión de trabajos

* Como operario quiero registrar un trabajo realizado para dejar constancia.
* Como agricultor quiero ver qué trabajos se han realizado en mis parcelas.
* Como responsable quiero filtrar trabajos por fecha y parcela.

---

## Épica: Usuarios

* Como administrador quiero crear usuarios para gestionar accesos.
* Como administrador quiero desactivar usuarios para controlar accesos.
* Como sistema quiero evitar acciones de usuarios deshabilitados.

---

## Épica: Consulta e histórico

* Como agricultor quiero ver el historial de mi explotación para tomar decisiones.
* Como responsable quiero consultar actividad por rango de fechas.
